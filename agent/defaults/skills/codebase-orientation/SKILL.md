# Codebase Orientation

Use this when a task needs project context.

1. Read the README, package metadata, and likely entrypoints.
2. Search for the exact symbols, commands, or files involved.
3. Prefer existing patterns over new abstractions.
4. Report the smallest useful implementation path.
